Get Started in Seconds

Open the editor or drop a few lines of code — either way, you're building games.

⚡ Quick Start

  1. Clone the repo and run npm install
  2. Start the dev server with npm run dev
  3. Click Launch Editor to open the visual editor
  4. Add GameObjects, attach components, and hit Play

📝 Code Example

import { WebForge, Vector3 } from './src/index.ts';

// Create the engine
const engine = new WebForge({ antialias: true });

// Add a game object
const player = engine.createGameObject('Player');
player.transform.position = new Vector3(0, 1, 0);

// Start the loop
engine.start();
